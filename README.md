## Project Setup:

**1. Create conda environment**
```bash
conda create -n venv python==3.8 -y
```
**2. Clone Github repository**
```bash
https://github.com/MannShrestha/Nuclei_semantic_segmentation.git
```
**3. Install requirements**
```bash
pip install -r requirements.txt
```
